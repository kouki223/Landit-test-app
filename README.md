# 環境構築
- リポジトリをclone
        - git clone https://github.com/kouki223/Landit-test-app.git
- 環境変数を.envに入力する
        - cp .env.example .env を実行して.envファイルを作成
                - .envファイル内にAPI keyを準備する
                        - GOOGLE_GEOCODING_API_KEY => 逆ジオコーディング用のAPI key
                        - NEXT_PUBLIC_GOOGLE_MAPS_API_KEY => => UI上で地図を表示するためのmap用API key
- ターミナルでリポジトリルートからスクリプトファイルを実行して環境構築が完了している事を確認する
        - ./check-setup.sh

# 実行手順
1. Docker 起動（DB/BE(API)/FE を一括でビルド・起動、CSVも自動インポート）
docker compose up --build
2. ブラウザで下記のURLを開く
http://localhost:3000
3. 期待する動作が行えるか確認する
- 地図を動かすと表示範囲のスポットが左に一覧表示
- 「半径検索」をONにして半径を指定 →「この範囲で検索」
4. 停止
docker compose down          # 停止（データ保持）
docker compose down -v       # DBも削除（次回起動でCSV再インポート）

# **実装時に特に工夫した点、および技術的な判断を行った箇所**
- 実装時に工夫した点
        - API呼び出し回数の削減
                - シナリオを考えてAPIの呼び出しを削減する事ができるシナリオに合わせて実装しました。
                        - より狭い範囲へズームした時
                        - すでにキャッシュされているデータの範囲の時
                        - 軽微な移動で逆ジオコーディングする必要がない時(geocode.service.tsにおいて、座標情報を小数点3桁以下にまるめる事で実現)
        - UX最適化
                - 自分がユーザーだったらどのようなアプリだと使いやすいか検討しました。
                        - 初期ロード位置は日本全体が表示される形が良い
                        - マーカーは中心固定だとわかりやすい
                        - クリックにマーカーが追従する形
                        - マーカーは全て表示すると範囲が広くなった時に見づらくなってしまう
                        - 半径検索はON/OFFができる方が半径表示のオーバーレイが消えるため見やすい
                        - マーカー or リスト表示部分をクリックする事でハイライトされる
        - Branch戦略に則った開発
                - 個人開発になるため、mainへの直Pushによる開発手法でも良いかと考えましたが、適切なBranch戦略をとって機能単位でBranchを切って開発しました。
- 技術的な判断を行った箇所
        - API keyを置く場所について
                - 環境変数などは、.envファイルで管理していますが、Google map API keyはフロント側で読み込んでいます。
                - API keyがフロント側で見えてしまう状態は良くない状態だと思いますが、API key側でプロキシを設定する方針にしたいと思います。（本番運用において）
        - useRefやuseEffectの副作用を扱うメソッドについて
                - 本来、useEffectなどの副作用を扱うメソッドに関しては、極力使用する事なく、本当に使う事が適切な時に使用する事が求められると思うのですが、useEffectやuseRefを代替するより良い解法の検証ができなかったため、useEffectやuseRefを活用する形をとりました。
                        - useEffectを使うと判断したメソッドはGoogle map API などを扱うメソッドとしました。
